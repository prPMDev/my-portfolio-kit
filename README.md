# My Portfolio Kit

AI-powered skills for building your personal portfolio with Claude Code.

**[Get Started](https://prpmdev.github.io/my-portfolio-kit/)** | **[See Skills](https://prpmdev.github.io/my-portfolio-kit/#skills)**

## What is this?

A collection of Claude Code skills that help you build a portfolio website. Not a template — you do the work, AI helps you move.

- **9 skills** for content, design, and quality checks
- **Builder voice** — shows what you shipped, not managed
- **Anonymization built-in** — protect work content while preserving impact

## Quick Start

```bash
git clone https://github.com/prpmdev/my-portfolio-kit.git
cd my-portfolio-kit
```

Then follow the setup guide: **[prpmdev.github.io/my-portfolio-kit](https://prpmdev.github.io/my-portfolio-kit/#quick-start)**
